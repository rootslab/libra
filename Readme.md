> _Libra_, a module to handle Redis commands/replies queues.

###Install

```bash
$ npm install libra [-g]
// clone repo
$ git clone git@github.com:rootslab/libra.git
```
> __require__ 

```javascript
var Libra = require( 'libra' );
```
> See [examples](example/).

###Run Tests

```bash
$ cd libra/
$ npm test
```
###Constructor

> Create an instance.
